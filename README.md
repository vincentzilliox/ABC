# ABC

Annoying Bioinformatic Curation

##Requirements

install git
```bash
yum install git
```

install [docker](https://www.docker.com/)

## installation

```bash
git clone https://github.com/vincentzilliox/ABC.git
```

## Start Services

```bash
cd ABC
./restart_services.sh
```
